KÝ HIỆU CÁC CỤM TỪ VIẾT TẮT
================================================================================

STT  Từ viết tắt     Nghĩa đầy đủ
───  ─────────────  ──────────────────────────────────────────────────────────
 1   API             Application Programming Interface
                     Giao diện lập trình ứng dụng

 2   BCA             Ban Công an (đơn vị tổ chức kỳ sát hạch lái xe)

 3   BGTVT           Bộ Giao thông Vận tải

 4   CCCD            Căn cước công dân

 5   CSDL            Cơ sở dữ liệu

 6   CSS             Cascading Style Sheets
                     Ngôn ngữ định dạng giao diện web

 7   HTML            HyperText Markup Language
                     Ngôn ngữ đánh dấu siêu văn bản

 8   HTTP/HTTPS      HyperText Transfer Protocol / HyperText Transfer Protocol Secure
                     Giao thức truyền tải siêu văn bản (bản thường / bản bảo mật)

 9   JSON            JavaScript Object Notation
                     Định dạng dữ liệu nhẹ dùng để trao đổi dữ liệu

10   JSX             JavaScript XML
                     Cú pháp mở rộng của JavaScript dùng trong React

11   JWT             JSON Web Token
                     Token xác thực theo tiêu chuẩn mở RFC 7519

12   MVC             Model – View – Controller
                     Mô hình kiến trúc phần mềm phân tách logic, dữ liệu và giao diện

13   MySQL           My Structured Query Language
                     Hệ quản trị cơ sở dữ liệu quan hệ mã nguồn mở

14   ORM             Object-Relational Mapping
                     Kỹ thuật ánh xạ đối tượng – quan hệ trong lập trình

15   OTP             One-Time Password
                     Mật khẩu dùng một lần (dùng trong xác thực)

16   PHP             Hypertext Preprocessor
                     Ngôn ngữ lập trình kịch bản phía máy chủ

17   RESTful         Representational State Transfer – compliant
                     API tuân theo kiểu kiến trúc REST

18   SPA             Single Page Application
                     Ứng dụng web đơn trang

19   SQL             Structured Query Language
                     Ngôn ngữ truy vấn cơ sở dữ liệu có cấu trúc

20   UI              User Interface
                     Giao diện người dùng
